<h1>minecraft_clone</h1>

<p>A basic procedural voxel terrain experiment built with the <strong>Ursina Engine</strong>.</p>

<h2>Features</h2>
<ul>
  <li>Procedural terrain generation with smoothed random heights</li>
  <li>Place and remove voxels with simple mouse input</li>
  <li>Easy to modify and extend</li>
</ul>

<h2>Essentials</h2>
<ul>
  <li>Python 3.8+</li>
  <li>Ursina Engine</li>
</ul>
    
<h2>Configure</h2>
<ol>
  <li>Clone the repository:
    <pre><code>git clone https://github.com/eard3n1/minecraft_clone.git</code></pre>
  </li>
  <li>Install Ursina:
    <pre><code>pip install ursina</code></pre>
  </li>
  <li>Run the app:
    <pre><code>python app.py</code></pre>
  </li>
</ol>
